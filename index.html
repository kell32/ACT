Aprendiendo el DOM

¿Qué es el DOM?

El DOM (Document Object Model) es una interfaz de programación para documentos HTML y XML. Representa la estructura de un documento web como una jerarquía de nodos, lo que permite que los lenguajes de programación interactúen con el contenido de las páginas web.

const nodo = document.getElementById('miElemento');

Manipulación del DOM con JavaScript

JavaScript permite interactuar con el DOM para modificar el contenido, la estructura y el estilo de una página web. Esto se logra mediante métodos como createElement(), appendChild() y removeChild().

document.body.appendChild(nuevoElemento);

Eventos en el DOM

Los eventos en el DOM permiten a los usuarios interactuar con la página web, como hacer clic en un botón, escribir en un campo de texto o mover el ratón por la pantalla. Los eventos pueden ser escuchados y gestionados usando JavaScript.

document.getElementById('boton').addEventListener('click', funcionClick);

Ejemplo de Creación de Elementos

Usar JavaScript para crear y agregar elementos al DOM es muy sencillo. Un ejemplo básico sería crear un nuevo párrafo y agregarlo al final del cuerpo del documento.

const nuevoParrafo = document.createElement('p');
nuevoParrafo.textContent = 'Este es un nuevo párrafo agregado con JavaScript';
document.body.appendChild(nuevoParrafo);
